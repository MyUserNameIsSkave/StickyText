# Changelog

All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-07-14

### Added

- Initial release.
- Scene view placement tool with three modes: Face Camera, Align to Surface, Fixed Distance.
- Coloured tag system driving label text/background colours and Editor Only defaults.
- Management window (Tools ▸ StickyText): searchable/sortable label list, tag manager, settings.
- Scene view overlay mirroring the most-used settings while placing.
- Configurable build stripping (None / Editor Only / All), separate for Development and Release builds.
- Mesh-picking or collider-raycast surface detection, with independent sample counts.
